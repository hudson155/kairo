# Kairo Image

Some utilities for working with images.

```kotlin
val jpeg = convertImage(png, "jpeg")
```

## Installation

Install `kairo-image`.

```kotlin
// build.gradle.kts

dependencies {
  implementation("software.airborne.kairo:kairo-image")
}
```

## Usage

### `convertImage()`

Converts an image to the specified format.

```kotlin
val jpeg = convertImage(png, "jpeg")
```
